## How to Run?
```bash
git clone https://github.com/golammonayem/US-Visa-Approval-Prediction
cd US-Visa-Approval-Prediction
conda create -n visa python=3.8 -y
conda activate visa
pip install -r requirements.txt
python app.py
```
